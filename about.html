University Project Notice

This website is part of a university coursework project and is not intended to display actual or authoritative information. All content is for educational and assessment purposes only.

Website Purpose

TechHorizon is an educational website exploring the theme of Innovative Technologies. It covers four major areas of cutting-edge technology: Artificial Intelligence, Space Technology, Quantum Computing and Biotechnology.

The website was designed and built to demonstrate skills in semantic HTML, CSS styling and responsive web design as part of the Essential Web module at WIUT.
